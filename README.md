# hello_rust

A simple Rust example project.

You can create a new Rust project using `cargo new project_name` (or `cargo init project_name`).

**Description:**
- This repository contains a small Rust application for educational purposes. The main code is located in the `src` directory.

**Requirements:**
- Rust toolchain (rustc + cargo). To install: `rustup install stable`

**Quick Start:**
- Build: `cargo build`
- Run: `cargo run`
- Tests: `cargo test`
